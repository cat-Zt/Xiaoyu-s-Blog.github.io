    # HTML

    ​ HTML (超文本标记语言) 构成了Web 世界的一砖一瓦, 定义了网页内容的含义和结构. CSS 则用来描述一个网页的表现与展示效果. JAvaScript 决定功能或者行为.

    # 网页的基本概念

    ​ “超文本”（hypertext）是指连接单个网站内或多个网站间的网页的链接。链接是网络的一个基本方面。只要将内容上传到互联网，并将其与他人创建的页面相链接.网页的HTML代码是由许许多多的不同的标签 (tag) 组成.通过“标签”（tag）将文本从文档中引出，标签由在“<”和“>”中包裹的元素名组成，HTML 标签里的元素名不区分大小写。也就是说，它们可以用大写，小写或混合形式书写。

    # 网页的基本标签

    ​ 不管多么复杂的网页,都是从这个基本结构衍生出来的.这个基本结构的主要标签,它们构成了网页的骨架。

    <!DOCTYPE html>
<html lang="zh-CN">
<head>
  	// charset 用来描述HTML文档的编码类型
     <meta charset="UTF-8" >
    // keywords，页面关键词：
     <meta name="keywords" content="关键词" />
    // description，页面描述： 
    <meta name="description" content="页面描述内容" />
    // refresh，页面重定向和刷新： 
    <meta http-equiv="refresh" content="0;url=" />  
    // viewport，适配移动端，可以控制视口的大小和比例
    <meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1">
  <title></title>
</head>
<body>
</body>
</html>

    • 标签 <!DOCTYPE html>表示文档类型，告诉浏览器如何解析网页。该标签采用完全大写的形式，以便区别于正常的 HTML 标签。因为<!doctype>本质上不是标签，更像一个处理指令
    • 标签 html 是网页的顶层容器，即标签树结构的顶层节点，也称为根元素（root element），其他元素都是它的子元素。一个网页只能有一个<html>标签。
    • 标签 head 是一个容器标签，用于放置网页的元信息。它的内容不会出现在网页上，而是为网页渲染提供额外信息。
      • 标签 meta：设置网页的元数据。--- 使用 name 以及 content 属性定义,用来 描述网页文档的属性 比如网页的作者，网页描述，关键词等
        • 标签 charset 属性 -- 指定网页的编码方式
        • 标签 http-equiv 属性 -- 用来补充 HTTP 回应的头信息字段，如果服务器发回的 HTTP 回应缺少某个字段，就可以用它补充。<meta>标签的content属性是对应的字段内容。这两个属性与 HTTP 协议相关
      • 标签 link：连接外部样式表。
      • 标签 title：设置网页标题。
      • 标签 style：放置内嵌的样式表。
      • 标签 script：引入脚本。
      • 标签 noscript：浏览器不支持脚本时，所要显示的内容。
      • 标签 base：设置网页内部相对 URL 的计算基准。-- 即跳转时会在base基础上进行跳转
    • 标签 body 容器标签, 用于放置网页的主体内容。浏览器显示的页面内容，都放置在它的内部
